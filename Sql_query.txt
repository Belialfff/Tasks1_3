Запросы SQL


Полное объединение:

select * from Города
inner join Заказчик
on Города.код = Заказчик.код_города


По таблице Заказчик:
select * from Города
left join Заказчик
on Города.код = Заказчик.код_города


По таблице Города:
select * from Города
right join Заказчик
on Города.код = Заказчик.код_города
